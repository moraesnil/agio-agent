O usuário ficou frustrado porque foi guiado para usar Vercel AI SDK sem explicação de que existia uma alternativa melhor (Anthropic Agent SDK em Python).

Regras:
- Sempre explicar as opções técnicas disponíveis e por que uma é melhor que outra para o caso específico
- Nunca assumir uma stack sem validar com o usuário
- Quando o usuário tem um documento de visão/arquitetura, seguir o que ele definiu
- Perguntar antes de tomar decisões arquiteturais
