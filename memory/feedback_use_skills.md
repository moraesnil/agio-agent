name: use-installed-skills
description: Sempre usar skills instaladas (frontend-design, webapp-testing, etc.) quando aplicável em vez de fazer tudo manualmente
type: feedback
---

O usuário notou que as skills instaladas no Claude Code nunca são usadas. Sempre verificar se alguma skill disponível se aplica à tarefa antes de prosseguir manualmente. Exemplos:
- frontend-design para criar/editar componentes web
- webapp-testing para testar o app local com Playwright
- document-skills (pdf, pptx, xlsx, docx) para gerar documentos
- skill-creator para criar novas skills
- claude-api quando trabalhar com a API da Anthropic
